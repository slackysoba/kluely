# pIqaD qolqoS

`pIqaD-qolqoS.ttf` is the "pIqaD qolqoS" typeface by Daniel Dadap
(daniel@dadap.net), from https://github.com/dadap/pIqaD-fonts
(release pIqaD-fonts-r1, v1.0).

It renders the Klingon pIqaD script using the ConScript Unicode Registry
encoding in the Unicode Private Use Area, code points U+F8D0–U+F8FF.
